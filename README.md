# E-Commerce Management System

Система управления товарами и категориями для интернет-магазина.

## Функциональность

### Реализовано в этом задании:

1. **Классы моделей**:
   - `Product` - товар с атрибутами: название, описание, цена, количество
   - `Category` - категория товаров с атрибутами: название, описание, список товаров

2. **Атрибуты класса Category**:
   - `category_count` - счетчик количества категорий
   - `product_count` - счетчик количества товаров
   - Автоматическое обновление при создании объектов

3. **Загрузка данных**:
   - Поддержка загрузки из JSON-файла
   - Автоматическое создание объектов из данных

4. **Тестирование**:
   - Полное покрытие тестами (>75%)
   - Тесты инициализации объектов
   - Тесты подсчета количества
   - Тесты загрузки данных
