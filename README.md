# E-Commerce Management System

Система управления товарами и категориями для интернет-магазина.

## Функциональность

### Реализовано в этом задании:

1. **Классы моделей**:
   - `Product` - товар с атрибутами: название, описание, цена, количество
   - `Category` - категория товаров с атрибутами: название, описание, список товаров

2. **Атрибуты класса Category**:
   - `category_count` - счетчик количества категорий
   - `product_count` - счетчик количества товаров
   - Автоматическое обновление при создании объектов

3. **Загрузка данных**:
   - Поддержка загрузки из JSON-файла
   - Автоматическое создание объектов из данных

4. **Тестирование**:
   - Полное покрытие тестами (>75%)
   - Тесты инициализации объектов
   - Тесты подсчета количества
   - Тесты загрузки данных

## Установка

## Новая функциональность

### Наследование классов товаров

Добавлены два новых класса-наследника от базового класса `Product`:

1. **Smartphone** - представляет смартфоны с дополнительными атрибутами:
   - `efficiency` - производительность
   - `model` - модель
   - `memory` - объем встроенной памяти
   - `color` - цвет

2. **LawnGrass** - представляет газонную траву с дополнительными атрибутами:
   - `country` - страна-производитель
   - `germination_period` - срок прорастания (в днях)
   - `color` - цвет

### Ограничения сложения товаров

Реализована проверка типов при сложении товаров:
- Можно складывать только товары одного класса
- При попытке сложить товары разных классов возникает `TypeError`

### Защита от добавления невалидных объектов

Метод `add_product` категории теперь проверяет, что добавляемый объект является экземпляром класса `Product` или его наследников.

### Тестирование

Добавлены comprehensive тесты для:
- Наследования классов
- Ограничений сложения товаров
- Проверки типов при добавлении в категории
- Функциональности новых классов
